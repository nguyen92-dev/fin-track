# Backend Expense Management

## Prerequisites

We use a custom library called restsqlbackend.
### To install restsqlbackend, run:
```bash
./mvnw install:install-file   -Dfile=restsqlbackend/rest-sql-backend-1.0.2.jar  -DgroupId=top.nguyennd   -DartifactId=restsqlbackend   -Dversion=1.0.3   -Dpackaging=jar
```
